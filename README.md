# Clash Pro

**Clash Pro** 是一个基于 Clash 内核深度定制的增强版客户端，内置多源公共订阅解析能力，致力于为用户提供 **开箱即用、零门槛** 的网络代理体验。

在保持原生 Clash 强大功能的同时，Clash Pro 对订阅获取、节点管理和使用流程进行了全面优化，让更多用户可以无需复杂配置即可快速上手。

---

## ✨ 功能特性

- 🚀 **内置多源订阅支持**  
  预置多个公开可用的订阅获取入口，自动拉取节点信息  

- 🔄 **自动更新机制**  
  定时刷新订阅内容，自动清理失效节点，减少人工维护成本  

- ⚡ **原生 Clash 内核**  
  完整保留 Clash 的高性能转发与规则能力  

- 🎛 **优化的节点管理体验**  
  节点自动分组，支持手动切换与智能选择  

- 🛡 **隐私优先设计**  
  所有配置与解析过程均在本地完成，不上传、不记录用户数据  

- 🎨 **简洁直观的 UI**  
  针对新手用户重新设计交互流程，即使是第一次使用也能快速上手  

---

## 🎯 项目目标

Clash Pro 的目标不是替代 Clash，而是 **降低 Clash 的使用门槛**：

- 不再需要到处寻找订阅链接  
- 不再需要研究复杂的配置结构  
- 不再因为节点失效而频繁手动维护  

让用户专注于使用，而不是配置。

---

## 👥 适合人群

- 想体验 Clash，但不想折腾配置的新手  
- 需要临时可用节点的普通用户  
- 希望即装即用的 Clash 爱好者

## ⚠️ 使用说明与免责声明

- 本项目仅提供 **客户端功能与订阅管理能力**  
- 内置订阅来源均来自互联网公开信息，Clash Pro 不提供、也不出售任何代理服务  
- 请用户在遵守当地法律法规的前提下使用本项目  
- 若某些订阅来源涉及版权或合规问题，请联系仓库维护者删除  

---

## 📦 安装与使用

1. 下载对应平台的最新版本  
2. 启动应用，默认订阅将自动加载  
3. 选择节点，一键连接即可使用  

> 进阶用户仍可自行添加私有订阅或本地配置文件。
